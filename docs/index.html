Les commandes de base

# man Doc et référence des commandes systèmes ```bash man man ``` `Entrée` pour faire défiler, `q` pour quitter Tips: marche aussi en recherche google
# Cat Concatène des fichiers vers l'output standard ```bash cat fichier1 fichier2 ```
# ls Liste le contenu d'un répertoire (le courant par défaut) ```bash ls . ls ~ ``` Questions : - Comment lister tous les fichiers d'un dossier en liste, du plus petit au plus gros, avec la taille lisible ? - Comment lister tous les fichiers d'un dossier en liste, du plus récent au plus vieux, avec la taille lisible ?
# alias Créer des alias de commandes (et les lister) ```bash alias alias my_ls="ls -larth" alias remote-ssh="ssh user@xxx.xxx.x.xx" ``` Questions - Que fait la commande `ll` ? - Créer un alias pour activer le VPN de AB Tips : les alias peuvent être chargés automatiquement en les ajoutant dans `~/.bash_rc` ou `~/.bash_aliases` (mieux)
